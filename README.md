# 🎤 Pak Debate Forum

A modern event registration and management platform for debate forums and competitions.

**Live:** [pakdebateforum.com](https://pakdebateforum.com)

---

## ✨ Features

- **Event Registration** - Easy registration forms for events
- **Admin Portal** - Manage events and view responses
- **Secure Login** - JWT authentication system  
- **CSV Export** - Download event responses as Excel files
- **Responsive Design** - Works on all devices
- **User Feedback** - Collect feedback from participants

---

## 🛠️ Built With

- Cloudflare Workers (Backend)
- Cloudflare D1 (Database)
- Vanilla JavaScript (Frontend)
- JWT Authentication

---

## 🔐 Security

Secrets and sensitive data are safely stored in Cloudflare. Never check in your credentials. See [SECURITY.md](./SECURITY.md) for details.
